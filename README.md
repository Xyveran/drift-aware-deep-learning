### Drift-Aware Deep Learning for Non-Stationary Sensor Data

This project builds on prior work in drift detection and classical ML baselines, extending the approach to deep learning and adaptive modeling. It explores deep learning approaches for handling distribution shift in real-world sensor data. It focuses on building models that remain robust under temporal drift through adaptation techniques and evaluation over time.

#### Structure

- `src/`         — core ML system
- `configs/`     — experiment configs
- `experiments/` — runnable pipelines
- `notebooks/`   — EDA
